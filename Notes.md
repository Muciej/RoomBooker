# User scenarios

1. I am an user and I want to book classroom for a specific time slot
2. I am an user and I want to check which classrooms are available for a given time slot
    - I do not need to be verified
    - my booking must be accepted by the administrator before showing in the system
    - it may be shown as "pending"
3. I am an administrator and I want to accept/deny booking made by the user
    - I need to authenticate myself with password and username to confirm that I have privilages to do that
